# 📊 Netflix Data Analysis & Dashboard

## 📌 Project Overview
This project analyzes Netflix content trends to extract strategic business insights using Python and data visualization tools.

## 🎯 Objectives
- Analyze yearly content growth
- Compare Movies vs TV Shows trends
- Identify top producing countries
- Study genre distribution
- Build an interactive dashboard

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## 📊 Key Insights
- Rapid growth observed post-2016
- TV Shows growing faster than Movies
- US dominates content production
- Drama & International genres most common

## 🚀 Advanced Analysis
- Year-over-Year Growth
- Genre Concentration Analysis
- Country Diversification Analysis
- Duration Trend Analysis
